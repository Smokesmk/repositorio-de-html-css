Tag img: Formas de uso

Siga os exemplos:

<img> Adiciona-se src para colocar a imagem, tese: precisa estar na pasta do projeto.

<img src="html-logo.png">

Obs: Algumas tags não possui tag de fechamento, ou seja, não tem necessidade, eu digo que não existe.

Outra coisa que podemos usar dentro da tag img é colocando o nome da pasta onde está inserido o arquivo, siga o exemplo:

<img src="img/html-logo.png">

Podemos usar também, endereços de links pela internet, não recomendo viu? Pois se cair o site, comrrompe o teu site.

<img src="https://discord.gg/mugiwaras">
